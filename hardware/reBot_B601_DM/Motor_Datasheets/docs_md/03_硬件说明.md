# 03 硬件说明

> DM-J4310-2EC V1.2 硬件接口与安装

---

## 物品清单

购买电机后，包装内应包含以下物品：

1. **电机（含驱动）** × 1
2. **电源（含 CAN 通信端子）连接线**：XT30(2+2)-F 插头连接线 × 1
3. **调试串口信号线**：GH1.25 连接线-3pin × 1

---

## 接口及线序说明

### 接口总览

| 接口名称 | 接口标注 | 说明 |
|---------|---------|------|
| **电源接口-1** | XT30(2+2)-F | 1. 通过 XT30(2+2)-F 插头的电源连接线连接电源，额定电压为 24V，为电机供电<br>2. 通过 CAN 通信端子连接外部控制设备，可接收 CAN 控制命令，反馈电机状态信息 |
| **电源接口-2** | XT30(2+2)-F | 3. 电机包含两个电源接口，任一接口可单独连接使用，也可多机串联使用，方便走线 |
| **调试串口-3** | GH1.25-3pin | 通过 GH1.25 连接线-3pin，使用 USB 转 CAN 调试工具（或使用通用 USB 转串口模块）连接到 PC，通过达妙科技调试助手对电机进行参数设置，以及固件升级等 |

### 接口详细说明

#### 电源接口（XT30 2+2）

**功能**：
1. 电源供电（24V 或 48V）
2. CAN 通信（CAN_H、CAN_L）

**特点**：
- 电机有两个相同的电源接口
- 可单独使用一个接口
- 可串联多个电机，方便走线

#### 调试串口（GH1.25-3pin）

**功能**：
1. 参数读写
2. 固件升级
3. 电机校准

**连接方式**：
- 使用 USB 转串口模块连接到 PC
- 波特率：921600 bps

## 指示灯状态

电机驱动板上有指示灯，用于显示电机当前状态。

### 指示灯说明

| 灯色 | 状态 | 说明 |
|------|------|------|
| **红灯常亮** | 失能状态 | 电机未使能，不接受控制命令 |
| **绿灯常亮** | 使能状态 | 电机已使能，可以接受控制命令 |
| **红灯闪烁** | 错误状态 | 电机出现错误，需要排查故障 |

### 错误类型（红灯闪烁）

当红灯闪烁时，可通过以下方式查看具体错误类型：
1. 串口打印信息
2. CAN 反馈帧中的错误代码
3. 调试助手显示的错误信息

**常见错误**：
- 过温保护
- 过压保护
- 欠压保护
- 过流保护
- 通讯丢失
- 编码器校准失败（错误类型 3）

## 接线注意事项

### 电源接线
1. **确认电压等级**：24V 版本使用 24V 电源，48V 版本使用 48V 电源
2. **注意极性**：正负极不可接反
3. **电源容量**：确保电源能提供足够的电流（峰值电流 24V 版本 16.5A，48V 版本 12.8A）

### CAN 接线
1. **CAN_H 和 CAN_L**：注意不要接反
2. **终端电阻**：CAN 总线两端需要 120Ω 终端电阻
3. **线缆长度**：建议使用双绞线，长度不超过 40m（1Mbps 波特率）

### 串口接线
1. **仅用于调试**：串口仅用于参数配置和固件升级，不用于实时控制
2. **波特率**：921600 bps
3. **连接时机**：可在电机上电前或上电后连接

## 多机串联接线示例

电机支持多机串联，方便走线：

```
电源 ──┬─→ 电机1 (接口1) ──→ 电机1 (接口2) ──┬─→ 电机2 (接口1) ──→ ...
       │                                      │
       └─→ CAN_H/CAN_L 总线 ─────────────────┘
```

**优点**：
- 减少接线数量
- 方便布线
- 统一供电

**注意**：
- 确保电源容量足够
- 每个电机设置不同的 CAN ID
