## 学习路径

1. **先看 URDF_入门指南.md** — 理解机器人描述文件的基本概念
2. **再看 MeshCat_可视化指南.md** — 学会用浏览器看 3D 模型
3. **然后去 `01_项目专用/`** — 看 reBot-DevArm 的 URDF 详解

## 前置知识

- 基本的 Python 使用经验
- 了解什么是机械臂（6 自由度、关节、连杆）
- 不需要懂 ROS / Pinocchio / 运动学
